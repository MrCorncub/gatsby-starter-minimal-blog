Youtube stellt mit der Youtube-iFrame API, die Möglichkeit zur Verfügen, Videos in die eigene Website einzubetten.
Der Code ist straight-forward und benötigt nicht viel Zeit, um ihn zu implementieren. Die Dokumentation befindet sich <a href="https://developers.google.com/youtube/iframe_api_reference?hl=de" target="_blank">hier</a>.

Es geht auch einfacher, indem ein iFrame-Element benutzt wird. Dieses Element bekommt das Attribut src, dass den Youtube-Link beinhalten soll.

```html
<iframe id="player"
     src="https://www.youtube.com/embed/NUYvbT6vTPs?enablejsapi=1&origin=http://example.com"
     frameborder="0"
></iframe>
```

Soweit so gut, jedoch gibt es noch ein einziges Problem. Das iframe-Element ist nicht responsive/dynamisch. Das Element soll sich automatisch
auf die Bildschirmbreite anpassen, das es zur Zeit nicht tut. Hier kommt CSS ins Spiel, dass den Style des Elements vorgibt und dafür sorgt, dass das Design
am Smartphone oder Computer gleich aussieht.

```html
<div class="videoWrapper" style="
        position: relative;
       	padding-bottom: 56.25%;
       	padding-top: 25px;
       	height: 0;">

  <iframe id="player"
        src="https://www.youtube.com/embed/NUYvbT6vTPs?enablejsapi=1&origin=http://example.com"
        frameborder="0"
        style="
            position: absolute;
            top: 0;
            left: 0;
            width: 100%;
            height: 100%;">
  </iframe>

</div>
```

Falls du es selbst ausprobieren möchtest und keine Website zur Verfügung hast, kannst du in dieser <a href="https://jsfiddle.net/omarjuvera/8zkunqxy/2/" target="_blank">Sandbox-Umgebung</a> den Code testen und verändern.
